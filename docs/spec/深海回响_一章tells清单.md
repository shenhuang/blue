## A. 四条 canonical tells（剧情 SPEC §4.1 锁定·全部已织入站内结局）

### T1 · 导师全程不摘面罩
- **位置**：`ch1.ending_station` body「在站里也戴着面罩——『热液场的规矩，』他说」；`lore.ch1.the_station`「他在记录架前，戴着面罩——热液场的规矩」。
- **表层（当时为真）**：热液场操作规范，戴罩合理。
- **回读指向**：主角**从未见过导师的脸**——接 `story_canon`「尸体脸永不可见」+ §4.4「主角在观测站『见到』的导师，真身在更深处」。面罩＝「不可见的脸」的具象。
- **永不交底**：表层「规矩」永远成立，永不确认面罩后是谁、是否还是人。

### T2 · 观测站干净得不像住了十年
- **位置**：`ch1.ending_station` body「站里很干净。十年的站不该这么干净」；**前置呼应**：`ch1.anchor_wreck` → `search_the_chart_room`「这屋子清得太干净了。在你之前，还有第三双手到过这儿」；`lore.ch1.anchor_wreck` / `lore.ch1.the_station`。
- **表层（当时为真）**：导师守站如守样地，器材归位、舷窗无盐霜。
- **回读指向**：站「不像被人住了十年」＝它未必是十年如一日被实体住着的地方（断片 / 亦真亦假）。沉船的「第三双手」把「有人先你一步清理过」的疑点前置——接 §4.4「失联=单程下行：先你而下的人」。
- **永不交底**：表层「他守得勤」成立。

### T3 · 队友只在余光 / 玻璃反光里
- **位置（站内）**：`ch1.ending_station` 两选项 outcome「器材舱里有人影在忙……每次却都差一个转头的角度——他们只在玻璃的反光里」。
- **位置（锚点前置伏笔）**：
  - `ch1.anchor_reef` → `wait_in_shadow` onFailure「一角织物，洗白的科考服那种蓝，在两簇珊瑚之间飘了一下，没了」；
  - `ch1.anchor_wreck` → `follow_the_rust_fish` onFailure「一顶旧潜水头盔，面窗朝你……里面当然是空的」；
  - `ch1.anchor_midwater` → `read_the_lights` onFailure「排成十年前码头送行那天的队形……队尾，没上船的那一个」（`lore.ch1.midwater_void` 同源）；
  - `ch1.anchor_vent` → `read_the_smoke` onFailure「一支科考编队……列队往更深处去」（`lore.ch1.vent_descent` 同源）。
- **表层（当时为真）**：氮醉幻觉 / 余光错觉（一章氮醉轴·`sanityRange` 低段·越深越重）。
- **回读指向**：队友「在场却永远看不真切」＝**尸衣者挂点**（§4.1「舷窗外/编队里的『队友轮廓』，幻觉还是穿着科考队装备的尸衣者——留给三章作钥匙」）+ 失联队伍的单程下行。midwater「没上船的那一个」另接主角**愧疚**（当年没跟船）的自我投射。
- **永不交底**：表层全是低 san 幻觉；尸衣者 / 失联读法只在回读叠加，二者始终并存。

### T4 · 导师「说」的每句话都出自日志已有句子
- **位置**：`ch1.ending_station` body「声音和日志里的批注一字不差」；`hand_over_the_log`「他说的每一句你都在日志里读过——他说出来，它们就都成了真的」「群落比坐标重要……写的人换一个，海不在乎」（= `lore.ch1.anchor_wreck` 页脚句 + 日志扉页句）；`look_around_first`「『人手一直够……』这句话日志的扉页上也有」。
- **表层（当时为真）**：主角烂熟导师日志，重逢时字句共鸣、倍感亲切。
- **回读指向**：「导师」**未提供任何日志之外的新信息**＝他可能只是主角**用日志拼出的投射**（断片说 / 氮醉放大执念）。「他说出来，它们就都成了真的」＝**主角自己在裁决真实**——正是留白结局录音那句注脚「氮醉会放大执念，永远不要被幻觉吞噬判断」。
- **永不交底**：表层「默契」成立，永不确认导师是否真在说话。
