## 1. 设计原则

1. **完整循环 > 全面教学**：所有核心系统都要走一遍，但每个只走一次最低限度。
2. **角色和世界一起教**：教程内容自身就是开篇剧情，不是脱离游戏的引导页。
3. **可失败但不致死**：玩家在这次下潜不应该真的死，但应该感受到几次"差一点"。
4. **结尾埋钩子**：埋下深海伏笔（某个看不清的轮廓、某句没说完的话），让玩家自己想要更深。

## 2. 角色与背景

### 2.1 主角（2026-06-12 改定·canon=剧情 SPEC §2）

无名（暂）**海洋生态科考潜水员**，写作避免指定性别。十年前，**导师**带队深海科考，整队失联；遗留**半本日志**＝全球潜点坐标串＋原创鱼/海底奇观草图＋批注。外界判定全队葬身海沟，**主角不信**。主角用着导师留下的装备（Aldo 保养多年），来到导师船队出发的鸢尾湾；科考不挣钱，先接公会的活糊口。

> 旧案「父母/祖父未归来的渔港之后」已废（canon 改导师）。「找尸体」伏笔不变——失踪者从家人换成导师队，挂点更深（归三章，一二章零暗示·quirk #117）。
> 导师人称暂用「他」（剧情 SPEC 只锁了主角不指定性别；导师人称待作者过稿一并拍）。

### 2.2 港口 NPC（教学关）

- **守灯人 Aldo**：港口酒馆 + 潜水公会的老人。脸上有大片伤疤。教学的"师父"。他是上一代探索过深处的人，因为某次事故再也不下水了。
- **打捞商 Mira**：年轻女商人，收购战利品。说话直率、会算账。
- **气瓶师 Otto**：沉默寡言的老技工。气瓶库的负责人。给玩家发初次潜水的装备。

教学关用 Aldo 一个 NPC 即可，Mira 和 Otto 在结尾出现一次。

## 3. 场景脚本

### 场景 1 · 港口序章（无下潜）

**展示**：港口 UI、装备槽、NPC 对话、出港选择。

```
[黎明前的鸢尾湾。雾还没散。一个穿黑色毛衣的老人坐在长椅上，看着海。]

Aldo："准备好了？"

> 准备好了。
> 还差一点。
```

**选项分支**：

- **"准备好了"** → 立即进入装备检查
- **"还差一点"** → Aldo 让玩家"先去港口看看"，可以参观气瓶库（看到 Otto 的脸）、打捞商（看到 Mira 的柜台），然后再回来

**装备检查（教学：装备槽）**：

```
Aldo："让我看看你的装备。

[气瓶]   蓝鳍 Mk.I        氧气 60 回合
[潜水服] 普通防寒服        体力上限 100
[灯具]   手提探照灯        视野中
[副手]   潜水刀            近战中等
[护身符] —                 未装备

你父亲的装备，我帮你保养了这么多年。
今天，你去东礁那个老沉船把这个浮标取回来。
那是公会的初次潜水，简单。

记住：氧气是时间，氮气是债。
慢慢上来，别贪。"
```

**Aldo 还会教玩家**：

```
"四样东西你要盯紧——
 体力，氧气，理智，氮气。
 前两样你能直接看到。
 第三样你自己感觉。第四样会找你算账。"
```

> 这句话埋了**理智是隐藏属性**的伏笔。玩家此时还看得到理智数字，但等深海的时候这数字会开始"撒谎"。

**选择下潜海域**：

教学关只开放一个海域 **"东礁"**（深度 5–35m）。点击进入下潜。

### 场景 1b · 出发——半本日志（开场钩 · 2026-06-12 新增 · `tutorial.prologue`）[已过稿 2026-06-12·整体认可]

**展示**：无新机制（纯开场拍）。**任务**：开篇剧情——「角色和世界一起教」的「角色」半边在这里立住：导师、半本日志、不信失联、科考不挣钱。日志是剧情，不是引导页。

实装：`zones.json` 的 `scriptedStartEventId` 从 `tutorial.descent` 改指 `tutorial.prologue`；两个选项**都**经 `setProfileFlags` 置 `story.ch1.hook`（engine/story.ts 的教学钩节拍位·dive 中持久写 profile·死在教学关不丢钩）并接 `tutorial.descent`——钩不依赖玩家选哪条。

```
出发前，你又把那半本日志翻开了一遍。
后半本不在了。泡掉的、撕掉的，还是导师自己取走的——十年了，你问过自己一千遍。

摊开的这页画着一条鱼：淡蓝的光从背鳍里透出来，铅笔描了三遍，像怕自己不信。
页脚一串坐标，一句批注：

「天黑了才找得见。」

档案里写：科考队全员失联，判定葬身海沟。
你不信。会留下这种字的人，不会就这么沉底。

今天的活儿和日志无关——东礁，老沉船，替公会把浮标取回来。
科考不挣钱；船和气瓶也不会自己修好自己。

你把日志塞回防水袋，咬上呼吸器。

> 收起日志，下水。
> 再看一眼那条鱼。
```

**设计注**（与剧情 SPEC 对齐）：
- 半本日志三件套照 §3.1：**坐标**（页脚）+ **原创鱼草图**（淡蓝背鳍光＝萤光背礁鳚·St1 锚点①预埋）+ **导师批注**（「天黑了才找得见」＝该鱼昼伏夜出的「去欺骗」提示·St1 兑现）。
- 「后半本不在了」三个候选解释并列、**不裁决**（规约 §5.8 永不交底）；为什么只剩半本＝一章背景悬念。
- 「描了三遍，像怕自己不信」＝导师的执念侧写——一章主题（执念）第一拍就在场。
- 红线自查：对「失联真相/断片说」零泄漏零暗示（「你不信」是 canon 明载执念·剧情 SPEC §2，不是机制暗示）；无尸体的脸；不触发 d_reveal；四锚点不在此露面（归 St1）。

### 场景 2 · 节点 1（深度 12m）— 下潜开始

**展示**：节点图、回合制氧气消耗、第一个无检定选项。

```
[12m / 氧气 58 回合 / 氮气 4 / 理智 100]

你顺着锚链下潜。光线还很亮，珊瑚礁在脚下伸展。
一群小鱼从你身边绕过。

> 继续下沉
> 停下来看看（消耗 2 回合）
> 沿珊瑚边缘游（消耗 1 回合 体力 -2）
```

**结算**：

- **继续下沉** → 直接到节点 2，氧气 −1 回合
- **停下来看看** → 氧气 −2 回合，**理智 +2**（恢复，体现"美景能治愈"）；触发简短描述：你看见一只小章鱼在岩缝里看你，它的眼睛比你想象的聪明
- **沿珊瑚边缘游** → 体力 −2，氧气 −1，**60% 概率拾到 1 枚珊瑚碎片（10 金币）**

教学要点：选项有不同代价/收益，没有绝对的"对"选择。

### 场景 3 · 节点 2（深度 20m）— 第一次属性检定

**展示**：体力检定、装备影响选项。

```
[20m / 氧气 55 回合 / 氮气 11 / 理智 100]

你贴着海底向北游，前方是一片倒伏的钢梁，
一只巨型石斑鱼挡在你和沉船之间，颚部张合着。
它有领地意识，但显然不是冲你来的。

> 缓慢绕开（屏息 / 氧气 -2）
> 用刀柄敲击金属吓走它（潜水刀 / 体力检定 12）
> 直接游近（强游 / 体力 -8）
```

**结算**：

- **缓慢绕开** → 一定成功，氧气 −2，进入节点 3
- **吓走** → 体力检定 12（玩家此时体力是满的，几乎必胜）：成功，鱼离开；**失败**：鱼短暂攻击，**体力 −5**，鱼后退
- **直接游近** → 体力 −8，触发"小型遭遇"事件 → 章节 4（变体）

教学要点：**装备让选项出现**（没有潜水刀的玩家看不到第二个选项）；**检定有可能失败但代价可控**。

### 场景 4 · 节点 3（深度 25m）— 沉船与轻量战斗

**展示**：第一次战斗、敌人姿态、应急上浮按钮（不需要按但要看到）。

```
[25m / 氧气 50 回合 / 氮气 18 / 理智 100]

沉船的影子覆盖了整个视野。
锈蚀的桅杆斜斜地戳在淤泥里，
你看见浮标——一个红白相间的圆筒，
卡在船舱的入口。

正要伸手时，一只暗礁鲨从船尾绕了出来。
它注意到你了。

[暗礁鲨 体力 50/50 警戒姿态]
```

**进入战斗**：

```
你的行动：
A) 鱼叉刺击   （体力 -4 / 氧气 -1）  ← 你没有鱼叉，灰显
B) 潜水刀挥砍 （体力 -6 / 氧气 -1）
C) 闪避       （体力 -8 / 氧气 0）
D) 屏息伏击   （氧气 -3，下次刀伤 +50%）
E) 投掷诱饵   （没有诱饵，灰显）
F) 屏息潜逃   （氧气 -3）
G) 应急上浮（必得减压病）
```

**预期玩家路径**：

- **D + B 组合**：屏息伏击 → 暴击挥砍，两回合解决，**鲨鱼体力归零撤退**（教学关不出现击杀演出）
- **F 屏息潜逃**：成功率 60%，失败则鲨鱼咬一口（体力 -8）后才得逞
- **直接 B B B**：3 回合搞定，但消耗较多

**关键演出**：

- 第 1 回合：鲨鱼从警戒进入攻击姿态（教姿态状态）
- 玩家拿下浮标时**画面提示**：氧气剩余 42 回合 / 安全上浮所需 6 回合（教派生预警值）

**战利品**：浮标 ×1（剧情物品）、鲨鱼牙 ×1（材料，价值 15 金币）

### 场景 5 · 节点 4（深度 30m）— 可选的"再深一点"

**展示**：贪婪的诱惑、第一次理智微跌、玩家的取舍。

```
[25m / 氧气 42 回合 / 氮气 25 / 理智 100]

你拿到浮标了。任务完成。
但船舱往下还有一个更暗的入口，
你的灯照进去，照不到底。

Aldo 说过别贪。

> 上浮（任务完成）
> 再深一点看看（深度 +5m）
```

**选择"再深一点"**：

```
[30m / 氧气 40 回合 / 氮气 36 / 理智 98]

入口里像是船长室。书桌、瓷器、一本被水浸透的航海日志。
你伸手想拿——

桌子后面似乎有个人影。

但那不可能，对吧？

> 抓起日志就走（氧气 -1）
> 看清楚那个影子（理智检定 80：成功什么都没有；失败：理智 -10）
```

**结算**：

- **抓日志走** → 拿到航海日志（剧情物品：解锁港口图书室升级线 + 之后某个深海事件链）
- **看清楚** → 这是玩家第一次看到"红字选项"提示之外的东西。检定大概率成功（玩家此时理智 98），但**有一行字会一闪而过**：

  ```
  ⟪ 那不是人影。但那也不是没有什么。 ⟫
  ```

  无论成败，**理智 −5**，并触发隐藏 flag `seen_first_uncanny`（影响未来某些深海事件）。

教学要点：玩家第一次接触到"再深一点"的诱惑——这是整个游戏的核心戏剧动作，必须早早建立。

### 场景 6 · 上浮序列

**展示**：上浮回合、减压停留、应急上浮的诱惑。

玩家选择上浮后，进入**上浮 mini-loop**：

```
[当前深度 30m / 氧气 38 回合 / 氮气 36 / 理智 93]

你开始上浮。

[上浮节点 1：25m] 安全
[上浮节点 2：15m] ▼ 减压停留建议 1 回合
[上浮节点 3：5m]  ▼ 减压停留建议 1 回合
[水面]

> 正常上浮（按建议停留）— 总耗 8 回合氧气
> 强行上浮（忽略停留）— 总耗 3 回合氧气，必得 II 型减压病
> 应急上浮（垂直窜出）— 总耗 1 回合氧气，必得 IV 型 = 直接死亡
```

**预期玩家选择**：正常上浮。

如果玩家偏要应急上浮，**Aldo 在港口会用沙哑的声音说**：

```
"我以为我说得够清楚了。
 上一个不听的人，装备现在穿在你身上。"
```

（2026-06-12 canon 注：旧台词「你父亲就是这么死的」与新 canon 冲突且从未实装——导师是失联不是确认死亡（剧情 SPEC §2 主角不信），Aldo 不会替档案下结论。改稿已随 2026-06-12 整体过稿、照现稿为准。）

然后**重开**教学关。（这是教程关，死亡惩罚降低，但语义保留。）

### 场景 8 · 教学关结尾——两本日志（2026-06-12 重写 · `tutorial.ending_log`）[已过稿 2026-06-12·整体认可]

**展示**：收开场钩（同一根钩，开场种、这里收——**别再另埋第二根**），引出下一次下潜动机。
触发不变：`item.captain_log.story.triggersEventId` → 回港 portEvent（无 run → flag 直接进 profile）。玩家在场景 5 选了直接上浮＝没有日志＝不触发本场、海图不开——重打教学关即可（脚本链不走随机池，`oncePerSave` 不拦；既有设计保留）。

```
夜里。你把捞回来的航海日志摊在桌上晾着。
大部分页面糊成了一块，能看清的只有一页。字迹很旧，是这条船的船长留下的——

————————————————————————
「……第七次见到它。
它不在这里。它从来不在这里。
但每次我下到 80 米以下，
它都『差一点』就要出现。

不是恐惧。是——
我不知道怎么形容。
像是在记起某件
我从来没经历过的事。」
————————————————————————

你把半本日志从防水袋里取出来，放在它旁边。
隔着不知道多少年，两个没见过面的人，写的是同一片海。

导师那页的页脚还是那串坐标。
你把明天要还给公会的浮标挪开，腾出桌角，摊开海图，
开始照着日志，把坐标逐条比对。

能落在这张海图范围里的，有四个。

航海日志剩下的页还粘着。急不得。

> 圈下四个坐标，合上日志。
```

**这个结尾收了四件事**：

1. **核心段落原文保留**（「第七次见到它……」），但归属改为**这条沉船的船长**（陌生人）——旧案「父亲的字」已废；
2. **两本日志互证**＝开场钩的收口：导师不是孤例，这片海写过同样的话——主角的「不信」第一次得到外部回声（仍不裁决真假·规约 §5.8）；
3. **「能落在这张海图里的，有四个」**＝一章四锚点（剧情 SPEC §4.1 表）的入场券：叙事上「抄坐标上海图」与机制上 `flag.tutorial_complete` 解锁海图是**同一个动作**——角色和世界一起教的「世界」半边在这里合龙；St1 把四个锚点 POI 真正放上去；
4. **「剩下的页还粘着」**＝软性回流锁点（§3.5 倾向自然暗示、不 UI 化明示），替换旧案「拿到合适的工具？」的显式机制许诺。
