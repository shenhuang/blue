# 深海回响 · 教学关剧本

> 第一次下潜：「初次潜水」
> 配套主 SPEC 与战斗系统 SPEC。
> 目标：在 10–15 分钟内让新玩家完整体验"出港 → 节点选择 → 事件 → 战斗 → 上浮 → 减压 → 回港 → 升级"全循环，并在结束时埋下深海伏笔。

---

## 1. 设计原则

1. **完整循环 > 全面教学**：所有核心系统都要走一遍，但每个只走一次最低限度。
2. **角色和世界一起教**：教程内容自身就是开篇剧情，不是脱离游戏的引导页。
3. **可失败但不致死**：玩家在这次下潜不应该真的死，但应该感受到几次"差一点"。
4. **结尾埋钩子**：埋下深海伏笔（某个看不清的轮廓、某句没说完的话），让玩家自己想要更深。

---

## 2. 角色与背景

### 2.1 主角

无名新晋潜水员。来自小渔港"鸢尾湾"（暂名）。父母（或祖父）是潜水员，**多年前未能从一次深潜中归来**。主角继承了家里的潜水装备和小屋，刚通过基础认证。

> 这个背景给"找尸体"机制做了天然伏笔——后续某次下潜可能遇到家人的遗骸。

### 2.2 港口 NPC（教学关）

- **守灯人 Aldo**：港口酒馆 + 潜水公会的老人。脸上有大片伤疤。教学的"师父"。他是上一代探索过深处的人，因为某次事故再也不下水了。
- **打捞商 Mira**：年轻女商人，收购战利品。说话直率、会算账。
- **气瓶师 Otto**：沉默寡言的老技工。气瓶库的负责人。给玩家发初次潜水的装备。

教学关用 Aldo 一个 NPC 即可，Mira 和 Otto 在结尾出现一次。

## 3. 场景脚本

### 场景 1 · 港口序章（无下潜）

**展示**：港口 UI、装备槽、NPC 对话、出港选择。

```
[黎明前的鸢尾湾。雾还没散。一个穿黑色毛衣的老人坐在长椅上，看着海。]

Aldo："准备好了？"

> 准备好了。
> 还差一点。
```

**选项分支**：

- **"准备好了"** → 立即进入装备检查
- **"还差一点"** → Aldo 让玩家"先去港口看看"，可以参观气瓶库（看到 Otto 的脸）、打捞商（看到 Mira 的柜台），然后再回来

**装备检查（教学：装备槽）**：

```
Aldo："让我看看你的装备。

[气瓶]   蓝鳍 Mk.I        氧气 60 回合
[潜水服] 普通防寒服        体力上限 100
[灯具]   手提探照灯        视野中
[副手]   潜水刀            近战中等
[护身符] —                 未装备

你父亲的装备，我帮你保养了这么多年。
今天，你去东礁那个老沉船把这个浮标取回来。
那是公会的初次潜水，简单。

记住：氧气是时间，氮气是债。
慢慢上来，别贪。"
```

**Aldo 还会教玩家**：

```
"四样东西你要盯紧——
 体力，氧气，理智，氮气。
 前两样你能直接看到。
 第三样你自己感觉。第四样会找你算账。"
```

> 这句话埋了**理智是隐藏属性**的伏笔。玩家此时还看得到理智数字，但等深海的时候这数字会开始"撒谎"。

**选择下潜海域**：

教学关只开放一个海域 **"东礁"**（深度 5–35m）。点击进入下潜。

---

### 场景 2 · 节点 1（深度 12m）— 下潜开始

**展示**：节点图、回合制氧气消耗、第一个无检定选项。

```
[12m / 氧气 58 回合 / 氮气 4 / 理智 100]

你顺着锚链下潜。光线还很亮，珊瑚礁在脚下伸展。
一群小鱼从你身边绕过。

> 继续下沉
> 停下来看看（消耗 2 回合）
> 沿珊瑚边缘游（消耗 1 回合 体力 -2）
```

**结算**：

- **继续下沉** → 直接到节点 2，氧气 −1 回合
- **停下来看看** → 氧气 −2 回合，**理智 +2**（恢复，体现"美景能治愈"）；触发简短描述：你看见一只小章鱼在岩缝里看你，它的眼睛比你想象的聪明
- **沿珊瑚边缘游** → 体力 −2，氧气 −1，**60% 概率拾到 1 枚珊瑚碎片（10 金币）**

教学要点：选项有不同代价/收益，没有绝对的"对"选择。

---

### 场景 3 · 节点 2（深度 20m）— 第一次属性检定

**展示**：体力检定、装备影响选项。

```
[20m / 氧气 55 回合 / 氮气 11 / 理智 100]

你贴着海底向北游，前方是一片倒伏的钢梁，
一只巨型石斑鱼挡在你和沉船之间，颚部张合着。
它有领地意识，但显然不是冲你来的。

> 缓慢绕开（屏息 / 氧气 -2）
> 用刀柄敲击金属吓走它（潜水刀 / 体力检定 12）
> 直接游近（强游 / 体力 -8）
```

**结算**：

- **缓慢绕开** → 一定成功，氧气 −2，进入节点 3
- **吓走** → 体力检定 12（玩家此时体力是满的，几乎必胜）：成功，鱼离开；**失败**：鱼短暂攻击，**体力 −5**，鱼后退
- **直接游近** → 体力 −8，触发"小型遭遇"事件 → 章节 4（变体）

教学要点：**装备让选项出现**（没有潜水刀的玩家看不到第二个选项）；**检定有可能失败但代价可控**。

---

### 场景 4 · 节点 3（深度 25m）— 沉船与轻量战斗

**展示**：第一次战斗、敌人姿态、应急上浮按钮（不需要按但要看到）。

```
[25m / 氧气 50 回合 / 氮气 18 / 理智 100]

沉船的影子覆盖了整个视野。
锈蚀的桅杆斜斜地戳在淤泥里，
你看见浮标——一个红白相间的圆筒，
卡在船舱的入口。

正要伸手时，一只暗礁鲨从船尾绕了出来。
它注意到你了。

[暗礁鲨 体力 50/50 警戒姿态]
```

**进入战斗**：

```
你的行动：
A) 鱼叉刺击   （体力 -4 / 氧气 -1）  ← 你没有鱼叉，灰显
B) 潜水刀挥砍 （体力 -6 / 氧气 -1）
C) 闪避       （体力 -8 / 氧气 0）
D) 屏息伏击   （氧气 -3，下次刀伤 +50%）
E) 投掷诱饵   （没有诱饵，灰显）
F) 屏息潜逃   （氧气 -3）
G) 应急上浮（必得减压病）
```

**预期玩家路径**：

- **D + B 组合**：屏息伏击 → 暴击挥砍，两回合解决，**鲨鱼体力归零撤退**（教学关不出现击杀演出）
- **F 屏息潜逃**：成功率 60%，失败则鲨鱼咬一口（体力 -8）后才得逞
- **直接 B B B**：3 回合搞定，但消耗较多

**关键演出**：

- 第 1 回合：鲨鱼从警戒进入攻击姿态（教姿态状态）
- 玩家拿下浮标时**画面提示**：氧气剩余 42 回合 / 安全上浮所需 6 回合（教派生预警值）

**战利品**：浮标 ×1（剧情物品）、鲨鱼牙 ×1（材料，价值 15 金币）

---

### 场景 5 · 节点 4（深度 30m）— 可选的"再深一点"

**展示**：贪婪的诱惑、第一次理智微跌、玩家的取舍。

```
[25m / 氧气 42 回合 / 氮气 25 / 理智 100]

你拿到浮标了。任务完成。
但船舱往下还有一个更暗的入口，
你的灯照进去，照不到底。

Aldo 说过别贪。

> 上浮（任务完成）
> 再深一点看看（深度 +5m）
```

**选择"再深一点"**：

```
[30m / 氧气 40 回合 / 氮气 36 / 理智 98]

入口里像是船长室。书桌、瓷器、一本被水浸透的航海日志。
你伸手想拿——

桌子后面似乎有个人影。

但那不可能，对吧？

> 抓起日志就走（氧气 -1）
> 看清楚那个影子（理智检定 80：成功什么都没有；失败：理智 -10）
```

**结算**：

- **抓日志走** → 拿到航海日志（剧情物品：解锁港口图书室升级线 + 之后某个深海事件链）
- **看清楚** → 这是玩家第一次看到"红字选项"提示之外的东西。检定大概率成功（玩家此时理智 98），但**有一行字会一闪而过**：

  ```
  ⟪ 那不是人影。但那也不是没有什么。 ⟫
  ```

  无论成败，**理智 −5**，并触发隐藏 flag `seen_first_uncanny`（影响未来某些深海事件）。

教学要点：玩家第一次接触到"再深一点"的诱惑——这是整个游戏的核心戏剧动作，必须早早建立。

---

### 场景 6 · 上浮序列

**展示**：上浮回合、减压停留、应急上浮的诱惑。

玩家选择上浮后，进入**上浮 mini-loop**：

```
[当前深度 30m / 氧气 38 回合 / 氮气 36 / 理智 93]

你开始上浮。

[上浮节点 1：25m] 安全
[上浮节点 2：15m] ▼ 减压停留建议 1 回合
[上浮节点 3：5m]  ▼ 减压停留建议 1 回合
[水面]

> 正常上浮（按建议停留）— 总耗 8 回合氧气
> 强行上浮（忽略停留）— 总耗 3 回合氧气，必得 II 型减压病
> 应急上浮（垂直窜出）— 总耗 1 回合氧气，必得 IV 型 = 直接死亡
```

**预期玩家选择**：正常上浮。

如果玩家偏要应急上浮，**Aldo 在港口会用沙哑的声音说**：

```
"我以为我说得够清楚了。
 你父亲就是这么死的。"
```

然后**重开**教学关。（这是教程关，死亡惩罚降低，但语义保留。）

### 场景 7 · 回港结算

**展示**：变卖、建设值、港口升级、NPC 反馈。

```
[东礁 · 初次潜水 · 已完成]

最深深度：30m
触发事件：5
完成事件链：1（沉船浮标）
本次战斗：1（暗礁鲨 / 击退）

获得：
  - 浮标 ×1（公会奖励 100 金币）
  - 鲨鱼牙 ×1（卖出 15 金币）
  - 航海日志 ×1（剧情物品，存入家中书桌）
  - 珊瑚碎片 ×1（卖出 10 金币）

港口建设值：+15
```

**Mira（打捞商）**：

```
"鲨鱼牙不错。
 多少钱一颗看心情，今天给你 15。
 下次拿好东西来，我们再算。"
```

**Aldo**：

```
"你比我想的稳。
 那本日志你自己看吧，我不读那东西。

 建设值你用来修港口。
 我建议你先升级船坞——
 出海能往更远的地方走，
 别老在东礁这种小地方泡着。"
```

**港口升级界面**：

```
建设值：15

[船坞] Lv.0 → Lv.1 （消耗 10）
  · 解锁新海域："旧灯塔礁"（深度 10–60m）
  · 出海前可多带 1 个消耗品

[气瓶库] Lv.0 → Lv.1 （消耗 12）
  · 氧气上限 +10 回合
  · 解锁备用气瓶购买
```

玩家无论选哪个，都能继续游戏（这是教学关，给一次"试错"机会）。

---

### 场景 8 · 教学关结尾

**展示**：埋下深海伏笔，引出下一次下潜动机。

玩家完成升级后回到港口主界面，**自动触发一次小过场**：

```
[夜幕。你坐在父亲的书桌前。
 航海日志摊开。
 大部分页面都泡烂了，但有一页能看清——]

————————————————————————
"……第七次见到它。
 它不在这里。它从来不在这里。
 但每次我下到 80 米以下，
 它都'差一点'就要出现。

 不是恐惧。是——
 我不知道怎么形容。
 像是在记起某件
 我从来没经历过的事。"
————————————————————————

[航海日志的下一页粘连得很紧，你撕不开。
 或许下一次……拿到合适的工具？]

[ 教学完成 · 你解锁了"旧灯塔礁" ]
[ 自由出海 ]
```

**这个结尾埋了三件事**：

1. **80m 以下有东西**——告诉玩家深海是危险的，但同时是有意思的
2. **下一页粘连**——一个具体的"以后能做"的动机（解锁某个工具就能打开）
3. **父亲见过类似的东西**——主角自己的故事不止于"接班"

## 6. 实施次序建议

垂直切片实装这份剧本的次序：

1. **港口主界面 + Aldo 对话**（场景 1）
2. **下潜状态机：4 个事件节点串成线性图**（场景 2–5）
3. **战斗系统最小实现：1 个敌人，4 个行动**（场景 4）
4. **上浮 mini-loop**（场景 6）
5. **结算与升级界面**（场景 7）
6. **结尾过场**（场景 8）

跑通这 6 步后，**整个 MVP 引擎几乎全部跑过一遍**。后续扩展 = 把"线性图"换成"随机节点图"，把"1 个敌人"扩成"敌人池"，把"1 个海域"扩成"多海域"——都不再是新系统，而是数据填充。

## 7. 待与你确认

1. **港口名 "鸢尾湾" / NPC 名 "Aldo / Mira / Otto"** — 这些是占位，是否要换成中文/其他风格？
2. **主角背景设为"继承父亲装备"** — 是否合适？也可以是"新人挑战者""遗孤""调查员"等其他切入。
3. **教学关是否允许真的死亡** — 我目前的设计是不死（应急上浮也只是重开），但你可以选择更硬核（让玩家真的体验一次死亡 + 尸体生成）。
4. **结尾日志是直接读 vs 玩家手动选择打开** — 影响沉浸感的细节。
