## 0. 一句话 + 与既有 canon 的关系
- **祭祀文明 ≠ 古文明**（作者 2026-07-07 锁）：古文明＝头足类（被膜拜的「神」）；祭祀文明＝**膜拜古文明的人类文明**。克苏鲁＝祭祀文明对古文明的误读（古文明 SPEC §0 既锁·本档承接）。
- **一句话**：一支祭祀文明的邪教把古文明「生物·蜂群」派当神，为求永生把海源寄生虫（**髓织虫**）用仪式强塞进自己体内——**成功了**：真永生、真变得像他们眼中的「神」、也真沦为傀儡。这是水鬼 B（人类版）的来路。
- **无共源护栏（钉死）**：髓织虫是**天然海源寄生虫·非古文明所造**；古文明只是被误读的信仰对象。人祸＝把天然虫**跨界塞进人**，不是「谁造了虫」。（守 §16.4 + 古文明 SPEC §6「水鬼＝多来路」。）

## 2. 髓织虫邪教（后祭祀文明中的一个小文明）
### 2.1 扭曲的神学
- 信仰的神**认可古文明「生物·蜂群」分支**（古文明 SPEC §2 四子派系·生物集体极＝「一个个体借上万具身体」）。漫长岁月里，他们对它的理解**早已扭曲**。
- 教义（扭曲后）：**只要能变成蜂群，就能重获神的认可 + 获得永生 + 被神接走。**

### 2.2 仪式（人祸跨界）
- 一小撮高层祭司**找到了髓织虫**（天然·**非造**）。
- **海源寄生虫本寄生不了陆生人类**——他们**创造条件、举行仪式、献祭族内的女人**（对位鱼版「宿主＝产卵母鱼」·虫认雌性·见 §16.1）。
- 髓织虫**控制不了庞大人体那么快**，有些实验以失败告终——但祭司认定这**恰是正道**，因为部分实验体**身体开始像恐人**；当时也会**撞见头足恐人**，被当作**古文明的化身**，于是被解读成**好兆头**。→ **§16.2「双重误分类」在历史现场上演**：把头足恐人当神谕、把半转化实验体当「走对了」。
- **接纳率 <100%·争相被选**：仪式是他们**费尽心思研究出来、用以提高「被接纳（感染）率」的技术**，即便如此也非人人成功。族人由此**三分**：入选的 / 想入却**落选**的 / **察觉不对出逃**的（出逃 / 清醒者留下**如实记录**·§5）。

### 2.3 接管与沦陷
- 终于髓织虫**适应了一具躯体** → 接管脊椎 → 大脑 → **慢慢同化协助仪式的祭司** → 最后整个小文明被接管。
- **成功即诅咒（dread 主柱）**：他们**真的得到了渴望的永生**、**真的变得更像他们眼中的「神」**（浅水里、人形、动得不对）。恐怖不在「他们失败了」，在**成功与沉沦无法区分**——与 §16.1「宿主是死是活无法定义」并列为本设定两根 dread 柱。接古文明自身之底：作弊硬上的头足也是「成了、也毁了」——**整张水鬼伞失败到底、没有赢家**。

## 6. 护栏（别塌）
- **无共源**：髓织虫天然·非古文明所造；古文明只是**误读的信仰对象**。别把虫算给「神」/ 古文明（否则 potato 塌·canon 过绑·见古文明 §6 / §16.4）。
- **无智**：髓织虫纯生物驱动·无意图；祭司的**恶意 / 狂热是「人」给的**·虫只管繁殖。Ch2 的「恶意」＝**人祸之恶 + 玩家投射**，不是虫有恶意（别滑进头足那种「有智悲剧」）。
- **reveal-lock（分层揭示·承揭示锁章纪律）**：
  - **Ch2 玩家面**只到**现象 + 人祸**（这些曾是人 / 他们对自己做了什么 / 别在台面摆寄生虫分类学）。
  - **「神＝古文明＝高科技文明」锁三章**（古文明 SPEC + 剧情反剧透）——Ch2 只留祭祀文明的**误读**（神＝抛弃他们的造物主），不接高科技底。
  - §4.2 既有「**天然畸变 + 氮醉**」＝玩家的**临时结论层**；本档 canon 是其**背后更深的真相**。如何在 Ch2 逐层交付（哪层给玩家、哪层留 dread）＝**Ch2 session 定**（§7），本档不预判。
