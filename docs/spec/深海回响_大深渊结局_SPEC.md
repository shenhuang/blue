## §1 一句话
玩家抵达大深渊底，与"导师"肉体决战 → **剧情杀**（导师杀死你的肉体、再追杀你的精神）→ **精神世界二战** → 你胜、导师精神被毁 → 从**导师的研究笔记**读到真相：**你从来不是人类**。你是古代文明（头足类）一次**相对成功的作弊**——有意识地占据并消耗人类躯体、靠古文明的复活点被"重新打印"。你之所以能无限复活，正是这个机制。**全程从没有任何一句叙述说过你是人类。**

## §2 前提 canon（就地重述 · pre-#300 · 待重建后 reconcile）
本终局建立在白板前的古文明/水鬼 canon 上。这些设计 doc 已随 #300 删除，此处只重述本文用到的骨干，主线重建时以重建版为准：
- **深渊锁＝世界法则**（引力·熵增·过阈才咬·非惩罚），不是谁的惩罚。三态出口：①留深 ②顺规退化（短命是后果）③作弊→水鬼（半成品）；**完好离开＝无解**。
- **湿件唯一**；文明动机＝**求知，非变强**，古今同。玩家的单程下行＝这个求知文明"能不能破法则"的最前沿原型。
- **水鬼＝尸衣者＝穿死者躯壳**（code `horror_sapien`）。
- **主角声线硬约束**：只写身体和环境，**从不定义主角是什么**（性质/性别/来历）。**断片说·尸体的脸永不可见。**

## §3 结局序列
- **§3.1 大深渊底·肉体战导师。** 终局关底，正常战斗。
- **§3.2 剧情杀（导师的攻击，不是叙事耍赖）。** 导师击杀你的肉体，随即要**追杀你的精神**——他知道你肉体死后精神仍在，因为他知道你是什么。死亡是他的**动作**，不是编剧的手，避开"打赢了却白白被杀"的坏手感。
- **§3.3 精神世界·二战。** 加一场战斗（**不依赖已删的理智系统**，就是一场换 register 的战斗；大深渊本就是加新料/换挡的地方）。你胜，**导师的精神被毁灭**。
- **§3.4 真相载体＝导师研究笔记。** 战胜后获得，读到他对古文明的理解 → 真相。接**收集碎片/观察手记**系列的现成呈现方式；真相是"读来的确认"，不是凭空砸下。
- **§3.5 揭示。** 你＝古文明·相对成功的作弊（见 §4）。玩家以为导师邪恶，最终发现**自己才邪恶**。

## §4 核心设定决策（讨论锁定）
- **§4.1 B 线·相对成功、非破法则。** 玩家是古文明"能不能破法则"的**最成功原型**，但**不是干净的完全成功**——"完好离开＝无解"不破：你并没有完好离开，你是一具**会被反复重打印的尸衣者**。一路上逃/杀的**水鬼＝你的失败前身、早期草稿**，你是唯一没散架的那个（回溯给全体敌人分量）。
  - 旋钮铁律：**"相对"必须有名有姓的代价**（见 §4.2）。只要玩家是干净全胜，世界法则的悲剧性就塌。
- **§4.2 代价＝夺舍，且必须消耗掉人类躯体**（人类绝无法接受的行为）。
  - **第一次夺舍吃掉了原主**（＝导师的那个学生）。一次性的大罪，"你才邪恶"由此有具体罪行垫底。
  - **此后的复活＝在古文明复活点"被重新打印"**（湿件/生物重塑，**不是机械 3D 打印**，接"湿件唯一"）。每次打印**退化一点**（记忆/自我侵蚀）＝**roguelike 每局从头**的 diegetic 解释。成功是一个**镀金的笼子**：法则没破，只是以更慢的方式在交学费。
- **§4.3 躯体漂移＝同一台机器。** 打印模板＝**第一具躯体的印记**→你的脸/身越来越像那个学生。这正是导师把你**当作学生**（后来又要杀你）的原因，也是最好的**早期伏笔**。「漂移 / 导师 / 先入为主」是同一根机制，别当三件事做。
- **§4.4 导师＝第一具躯体原主的导师，玩家先入为主继承了这段关系。** 你从来没有过导师——你接手了一个死者的关系网。导师**知道你是什么**（所以要杀你的"精神"），因此他有一个**悲剧动机**：他不是恶人，是一个眼看学生被一个顶着学生的脸、往深渊里钻的东西取代了的人。这一刀**回溯重写全程每一次师徒互动**（他一直在对他真正的学生说话，你这寄生物顺势接了下来）。
- **§4.5 复活机制两段 + 铁律「揭示≠变身」。**
  - 两段：**港口复活**（前期）→ **古文明复活点**（大深渊·须解锁）；后者不再出现在港口，而是**被重新打印**。
  - **铁律（本版命门）：大深渊是"揭示"你一直是什么，不是把你"变成"古文明。** 机器从你**第一次碰它就照用不误、你一点没变**，因为你本就是古文明。若玩家读成"进了深渊、用了机器、于是变成古文明"，整个"从没说过你是人类""无限复活也因为你是古文明"就塌成普通的"潜水员在深处变怪物"。
  - **前段缝：港口那些复活要能被结局回收。** 口径：港口复活**一直就是同一套古文明机制**，只是被你（和玩家）当成理所当然的"重生"；大深渊只是让它露出原形、露出源头。这样重读时港口复活全都成立、不穿帮。
- **§4.6 公平伏笔＝"人类用不了、你却能用"的复活点。** 古文明复活点 canon 上**人类无法使用**，而你用上了——这台机器替玩家"我是人类"的假设打了个叉，且**不靠叙述断言任何东西**（与 §5.2 铁律相容）。
  - **剂量：先用、后懂。** 别在你使用之前就把"人类无法使用"写在脸上（那等于报答案）。让你先莫名其妙地用上（玩家自找借口：运气？深潜把我改了？），"没有任何人类记录能启动它"这类话留到之后**从导师笔记里读到**，才有慢慢反应过来的后劲。

## §5 不穿帮机制（fair-play twist）
- **§5.1 两种死法，方向相反。** 提前泄底（玩家太早猜到，reveal 变平）vs 前后矛盾（前面写死"你是人类"，reveal 变耍赖）。同时躲开的唯一办法：**前面每条线索，重读时全都成立、初读又能被解释成别的**（double-coding）。
- **§5.2 铁律：假设≠断言（落成 lint 门）。** **叙述体永不断言主角的人类身份**；玩家/NPC 的**先入为主可以**（NPC 叫你那个学生的名字、你默认那是你的名字）——叙述自己的口不能替这个假设背书。
  - 机制化：`check-no-human-assertion` regress 门（本次实装·见 §7）扫叙事文本里"叙述体断言主角是人类/人类解剖/我们人类"的违规。**白板期正是立门的最佳时机**：新正文从第一天就受约束，远胜事后审漏台词。
- **§5.3 双读伏笔清单（每条都要能两读）。** 身体在变（读成深潜/减压把你弄坏了）· 无限复活（读成"深海不让你安息"的谜）· 导师冷硬（读成师徒心结或他才是反派）· 镜中脸 / NPC 的迟疑 / 一张对不上的旧照 / 你"醒来"而不是"重生"的措辞。
- **§5.4 decoy：让玩家怕自己"变成水鬼"。** 等 reveal 一到"你从一开始就是"，这份怕直接反转。
- **§5.5 与 #297 结局2（Théo 水鬼）的镜像。** Théo 在 Ch1 变成水鬼，**预演/映射**玩家终局揭示的自身本质；两种水鬼成因（Théo＝人祸派生 / 玩家＝古文明原型）**非冲突、可协同**——Théo 的转变是给玩家结局埋的一层情绪伏笔。
