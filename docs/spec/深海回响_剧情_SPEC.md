# 深海回响 · 剧情定调 SPEC

> 状态：定调 v1 · 2026-06-11 作者拍板（决策记录见 §6）。
> 本档 = 叙事 canon + 开放问题登记处。机制细节归各自 SPEC；本档只锁方向与口径。
> 改动原则：调性级改动须作者拍板；潜点/鱼种等内容级可由 session 按 §5 规约扩充。
> 与主 SPEC 的关系：理智/氮醉机制底座已在主 SPEC §3/§5（氮气→氮醉→扣理智·红字选项/假节点/UI 故障）——本档不加系统，只给底座**换弹药**：把泛克苏鲁幻觉重新校准为「执念具象化」。

## 0. 一句话内核（叙事北极星）

**深海还你你带下去的东西。** 氮醉不制造幻觉，只放大你携带的东西——执念（一章）、恐惧（二章·且不是你的，是替死去岛民背的）、愧疚（番外）。

- 与机制北极星「越深越欺骗」的关系：**欺骗 = 环境欺骗（mimic/穿尸者/猎手/地形）× 心理欺骗（氮醉）**，双轨暧昧：实存生物照旧实存，氮醉怀疑罩在一切之上。**角色可以下结论「是氮醉」，游戏永不交底。**
- **「环境越空，脑子越满」**：越深，真实环境越稀疏抽象（这条「越深越稀疏」的梯度不变；具体深度分层已从 abyssal→nameless 旧 band 改为数据驱动「深度柱」，见 `深海回响_探深深度柱_SPEC.md`），氮醉幻觉越喧哗。两条曲线交叉而非同步。
- 科考 = 反复下潜是职业常态：roguelike 循环无需叙事解释。

## 1. 认知阶梯（章节总构）

相信幻觉（一章圆满）→ 看穿幻觉（一章留白·二章真相）→ 驾驭幻觉（三章·主动入幻不迷失）。番外在阶梯之外：全游戏唯一无反转章节，攒叙事信任。

**机制阶梯（与认知阶梯孪生·决策 11）**：一章幻觉=**干扰** → 二章幻觉=**实化**（低理智下敌人行为变化、事件解锁不同方向——幻觉影响现实）→ 三章幻觉=**驾驭**（刻意入幻）。二章实化是一三章之间必需的台阶：幻觉先从「骗你的」变成「有用的」，「主动入幻」才有学习曲线。番外引入正交生存轴=**温度系统**。每章机制落地配「回流解锁」内容批（§3.5）。

**章节五槽（2026-06-12 八批对齐·以此为准；本档旧「三章」字样多指今「四章」，主 SPEC §11 章节制重写时全文回校）**：
一章=主要机制+科考队剧情 → 二章=精神/毒性等机制+祭祀文明剧情（真结局引大深渊情报·六批）→ 外传=冷热环境机制+摄影师剧情（结束后主线主角可回流探索）→ **三章=进渊预备**：收集探索资格（科技=破极限装备 vs 基因=进化鱼鳍——**在此选边，重复古文明当年的分裂**；回流一二章/外传收线归此章）→ **四章=无回头大深渊**（倒置规则/分层向下/会师层归此章）→ 五章=留白（古文明想知道的「更多」·到时候再定）。
认知弧对应：一章写实（幻觉=氮醉）→ 二章超自然涌现 → 三四章解咒（神=高科技文明）→ 再反转（可能真有造物主）→ 五章再上。

- 解锁链：一章圆满 → 二章；一章特殊道具 + 二章特殊剧情 → 番外；三章 = 回流一章四场景（更强关底）→ 二章潜点收线 → 学会入幻 → 「另一个世界」。
- 各章随时可回流（更难变体、刷材料）。
- 词汇隔离：**叙事「章」≠ 机制「Phase」**。三章（叙事）对接深水区 Phase 3「另一个世界」——**方向已定调（§4.4·2026-06-11 二批）**，机制/数值/内容设计仍留专门 session（d_reveal 不动）。
- 现有 300m+ 沉没之城（隐藏 BOSS·理智不可恢复）= 三章的门，即提案「城市遗骸」，已在主 SPEC §5.2 就位。

## 2. 主角与事实账本（防跨 session 漂移）

- 主角：海洋生态科考潜水员，无名（暂），写作避免指定性别。
- 主角语调（voice·单一来源·防漂移）：**冷静寡言**；情绪只走**不自主的身体反应**（手抖、耳鸣、呼吸节奏、喉咙发紧），不直白命名情绪 / 不戏剧化生理——禁词：感到害怕、心跳加速、鼻子一酸、眼泪/泪水/流泪/落泪/哭了/大哭；死亡不戏剧化（不用「命悬一线」类）。**禁词由 `scripts/check-protagonist-voice.mjs`（regress 门·quirk #184）守**（扫事件正文/选项/日志页 lore·NPC 台词豁免）；其余语调（零机制解释、细节优先于概括）属写作评审指引。洞穴池历史表述见 `cave_zones_spec.md`「全局语调约束」——以本条为准。参 [[protagonist-voice]]。
- T-10y：导师带队深海科考，整队失联；遗留**半本日志** = 全球潜点坐标串 + 原创鱼/海底奇观草图。外界判定葬身海沟，主角不信。
- 当下（一章）：循坐标独潜逐一探查。
- 一章后（二章）：科考不挣钱，缺钱接打捞委托——封禁百年孤岛·殖民时代宗教祭祀船·委托方只给残缺航海图、隐瞒诅咒传闻。
- 番外主角：**独立可玩角色**·创伤潜水摄影师（原「方案三」主角案独立成角）。T-3y 搭档遭离岸暗流失踪，因慌乱提前上浮，自责至今；心理医生建议重走两人约定的环球潜点。**开局比一章主角强**（额外装备/科技起步），面对的困难也更复杂；**不继承主线主角任何资产**。番外通关后，主线主角可进番外地图探索/刷材料。两人可在某潜点擦肩（彩蛋级，不强耦合）。摄影师救赎之旅与主线大致并行；主线主角回流番外图时**人已不在场**，只剩痕迹（回流叙事见 §4.3）。
- 录音：一章留白结局得到的水下录音是贯穿三章的线索；三章解锁更多分段（§4.4）。

## 3. 叙事系统（与现有机制映射）

### 3.1 导师日志（揭示性信息基建）
日志页 = 收集品 + 信息道具，对接现有信息轴（灯塔 reveal / 海图 POI / 声呐）。每页：坐标（解锁海图剧情锚点）+ 原创鱼草图（图鉴预填）+ 导师批注（该潜点的「去欺骗」提示）。
二章对位物 = **祭祀石板（封印性信息基建）**：同为前人留下的信息，日志说「去看」，石板说「别看」。两章对「该不该知道」相反作答——保持这个对仗。

> **剧情碎片可查阅（作者拍 2026-06-12·#118·实装排期）**：日志页等剧情碎片获得后**作为可查阅的收藏留存**——玩家随时回看。实装含三件：lore 正文登记表（现 `loreEntry` 只是自由 id 串进 `profile.loreEntries`，叙事正文散在事件 outcome 里读完即逝——需 `data/lore.json` 类 id→{title,body} 登记）+ 日志查阅 UI（港口或随时·按章/按潜点分组）+ lint「事件引用的 loreEntry 必须已登记」（机制化·照 check-event-dc 先例）。与 §3.2 录音「固定下来的=可信」同口径：写下来的字不随氮醉漂移，日志收藏就是玩家的真值锚之一。回读性也正是圆满结局 tells（#117）的兑现通道。

### 3.2 录音 = 真伪锚点
口径：**固定下来的声音（录音）可重放、不随氮醉漂移 = 可信；实时所见所闻 = 可疑。**「深海回响」题眼落在这。
机制候选：深处重放录音 → 稳理智一拍（幻觉变体退场/理智回升小额）。细节并入理智系统扩展（§8 工单③）。
**固定影像（照片）同口径。** 由此摄影师的相机是天然「验真」装备候选：拍摄=把可疑的实时所见固定下来核验（以胶卷/冲洗成本平衡强度）——番外回馈主线的镇章道具候选，亦是三章「入幻不迷失」的锚点候选（§7 登记）。

### 3.3 图鉴（自带欺骗轴）
先收录「你看到的版本」（可能是氮醉变形），后续观察修正条目。
原创鱼规约：**一站一种 · 真实物种打底 · 幻觉变体从原创鱼长出**。喂周末引擎：未来开 [氮醉] 变体批次（先例 [abyssal]/[reef]）。

### 3.4 氮醉（对接现有系统，不另起炉灶）
主 SPEC 已有：氮气 0–100（深度累积）→ 高浓度+深度 = 氮醉扣理智；理智阶段表（红字选项/假节点/UI 文字故障/疯狂上浮）。
剧情侧映射：幻象递呼吸器 = 红字选项；虚假坐标引向盲区 = 假节点；气量误读 = UI 故障。**幻觉只产错误信息，选择永远在玩家**（无脚本死口径覆盖叙事）。
扩展候选（并入主 SPEC §5.1 口径，另立机制小节）：低理智档仪表/罗盘读数漂移、录音锚定。
科学口径 = 诗意压缩：深度数值、氮醉统称都不按真实生理纠偏——后续 session 别「好心修正」。

### 3.5 回流解锁与解锁高亮（决策 11）
- 新机制/能力/道具落地后，老区域埋的对应锁点亮起：选项**解锁高亮** + 括号标注解锁原因（如「（相机）」「（耐温服）」）。
- **从数据渲染，禁手写**：高亮与括号原因由选项的结构化解锁字段（`unlockedBy` 类）渲染，label 不手写——沿 #109 check-event-dc「label 禁标注」先例，实装时配 lint。
- **配色语义隔离**：解锁高亮 ≠ 选择高亮 ≠ 红字幻觉诱饵，按解锁来源大类各配独立色（道具/能力/剧情进度…类目实装时定稿）。**红字是骗你的，解锁徽章永远诚实**——它从 profile 数据渲染，是玩家在欺骗世界里的机制锚点（同录音口径）；三章若想打破此信任，作者在场逐拍，默认不破。
- 锁点呈现（解锁前）：倾向自然暗示、不 UI 化明示「此处有锁」（守欺骗轴的克制）；实装时定稿。

### 3.6 死亡口径=断片（决策 12）
- 死亡永远写成**断片**：你总在港口醒来，浑身减压伤，记不得最后一段；下面多了一具穿你装备的尸体。
- **一二章对死亡机制零解释、零暗示**（写作规约 10）。表层读法自洽：氮醉断片+这片海本就满是失踪者（导师队是先例），「自己的尸体」可能是你认错了。深层读法留三章翻牌：**「死亡后从更深处开始」不是 respawn 规则，是真相**——留在下面的部分没有消失，一直在往下走；回港的是「还能回去的那部分」。死得越深回去的越少，三章某层之后再没有能回去的部分＝层级单程的机制化解释，与失联真相（决策 7）完全同构。
- 自动充能的既有件：#69 diverName=「你」、穿尸者的温柔陷阱、留白录音的警告（多一层读法：别在深处死太多次）。
- 零机制改动：现有 死亡→回港→尸体回收/葬礼 系统原样保留，只改文案口径。揭示节奏归三章 session。

- **圆满结局（canon：主角以为的真结局）**：海沟底找到水下永久观测站，导师为守护未被发现的深海生态圈长期留守，日志是留给后辈的生态档案，原创鱼在此成完整群落，主角完成科考传承。**按「当时为真」呈现**——主角此时未解锁隐藏真相，玩家第一遍也不该被戳穿。文本埋**低调 tells 供回读**（清单已定稿·单一来源见 [一章 tells 定稿清单](深海回响_一章tells清单.md)）：导师全程不摘面罩；观测站干净得不像十年；队友只在余光/玻璃反光里；导师「说」的每句话都出自日志已有句子。真假裁决权留给三章（隐藏结局）。

  - **「清醒/更难」的机制归二章**（Q&A 2026-06-21·机制按需长出）：一章不做全局「够深必出幻觉」（会逼玩家永久佩戴税）；改为**幻觉 scoped 到本事件 + 二章多处**，破损饰品**用宝石材料修复**成真·抵消幻觉锚点（§3.2 录音重放的姊妹件）后才能续二章剧情——见 §4.4「饰品/录音锚点」块。一章录音段为第 1 段（分段 canon §4.4·2+ 段留二章）。
- 穿尸者挂点：舷窗外/编队里的「队友轮廓」，幻觉还是穿着科考队装备的穿尸者——永不交底，留给三章作钥匙。

### 4.4 三章（进渊预备）+ 四章（大深渊）· 深渊与文明（二批起定调·八批对齐五槽 · 机制/数值/内容设计仍留专门 session）

> 本节早期内容按旧编号写「三章」——八批后口径：**资格收集/双流派选边=三章；倒置规则/分层向下/会师层=四章**（§1 五槽）。字样回校随主 SPEC §11 重写一起做。

**入幻前置（原开放问题·已答）**：二章探索结束后，主角被引导**刻意压低理智并保持**，以此解锁/收集颠覆性造物。此前教玩家躲的「红字选项」就是门的机制位——先教你躲，再教你认路，回溯重估前两章的全部玩法经验。

**颠覆造物**：水下呼吸之鳍（氧气总量不再是约束）、超越人类极限的耐压物等。逐项删除旧资源约束——删一条旧约束，必须由三章新约束接管（见下）。

**超级大深渊与文明遗骸**：深渊极深、往里走更深——**永不见底 = 永不交底的地理形态**。深渊周缘是被遗弃的高度发达文明城市残骸。**该文明 = 二章祭祀文明信奉的「神」**：古人不理解科技，把它当作了神。二章真相结局留的缝在此接上（封印对象有了下落），但**解释的接力不止步**——文明自己也必须留下无法解释的问题（为何遗弃城市？它们面对的更深处是什么？），见规约 9。

**倒置生存规则（方向锁定·数值后定）**：
- 地图结构随之切换：前几章同图分区解锁，三章起改为**分层向下**（层级制·决策 8）。
- 氧气/氮醉不再是问题；**上升要付出巨大代价**；每下一**层级**即单程——到达该层即明确提示无法返回，再也无法活着回去；**死亡后从更深处开始**。
- 游戏语法整体倒置而押韵：之前死亡送你回港（上面=家），三章死亡往下送（下面=家）。**游戏不再送你回家，是三章最安静的恐怖。**
- 单程层级必须是**知情选择**（明确告知的不可回点）——无脚本死口径的延伸：不可逆要玩家亲手按下去。
- 候选对应表（专门 session 定稿）：港口↔遗骸前进基地；灯塔 reveal↔重燃文明设施；减压规划↔上升代价；死亡回港+尸体系统↔死亡下沉（**你历次的尸体都在比你更深处**——与穿尸者天然化反）；鱼类图鉴↔文明器物档案。

**人鱼互走（canon 一句话）**：前两章是海里的东西像人（人脸鳕群/人头鱼/人手薄膜），三章起是人慢慢像海里的东西（装鳍、弃氧、下行）——**越往下，人与鱼从两头互相走近**。穿尸者穿潜水员，潜水员穿深渊。
**已升级为机制方向（决策 11）**：主角随下潜深度增加**渐变像鱼**——颠覆造物是台阶，变鱼是轴。候选联动（三章待答）：变鱼进度=「氧气位」资源的反面（人性余量）；上升代价的形态来源（越像鱼，水面越回不去）。

**不可靠余地（规约 8 延伸）**：文明证据链按实存写（它要承担解释二章之神的功能），但叙述永不排除另一种读法——鳍真的有效，还是入幻者最后的叙事？**不需要做实，只需要不否认**，与一章圆满结局同款口径。

**预埋钩子（一二章现在执行·不变）**：

**大深渊双分支会师层（2026-06-12 七批·讨论级拍板·远期·不进交接 prompt）**：进渊章（六批章节分层后=四章·本档章名待 §11 重写时统一回校）发现古文明分裂为两支——**生物基因融合**（引入「灵力」：法术样能力·与使用者灵魂相关）与**科技**（引入 AI：自动处理+辅助战斗）——即玩家双流派（同化海洋生物/接纳古文明科技·六批 canon）所重复的那次分裂。后续揭示：AI 的类意识=仿生理论堆叠的涌现，**没人知道为什么这种结构堆起来会产生意识**。真相一层：这种结构本身触发对「灵界」的**链接**——造物主的法则：生物大脑=原生设计，文明计算单元=仿制；其它科技线也呈同趋势。双重反转收束：祭祀文明拜错对象（神=文明·已揭）→ 文明自己在接近一个它没拜的东西——**可能真有类似造物主的存在**，答案在大深渊更深处。

**八批精确化（2026-06-12·世界观对齐）**：链接结构=**神经元结构**——肉体神经元=原生设计，算法神经网络=仿制，结构对了就链接、无关载体；**AI 意识=真实意识**（涌现之谜的谜底=链接；现实 NN 类比只作设计语言，游戏内走文明文献口吻——「神经元结构」可说，影射不点破）；**深渊本土生物神经元结构更强**→不仅有意识还能外显灵力（候选：mimic/穿尸者等 apex「邪门」的统一世界观底=神经元强度梯度）；**造物主分层**——碳基生命起源≈**前一任**造物主之作，「物质达量级涌现生命」等法则的创造者可能更上层（台阶向上无封顶·交底台阶化：机制可交底、顶端永不）；**古文明下行动机闭环**=研究本土生物+AI→读懂法则→想知道更多→不断下探——**他们带下去的执念是求知欲本身**（内核「深海还你你带下去的东西」的文明级实例·与失联=单程下行同构）；五章=留白。

**三章 session 待答**：三章的「氧气位」资源是什么（候选：理智/自我本身，或=人变鱼进度的反面「人性余量」——氧维持身体活着，新表维持「你还是你」）；上升代价的形态（候选：变鱼进度·越像鱼水面越回不去）；「被引导」的引导者是谁（录音？日志草图？文明信号？——失联真相落定后，**导师日志的后半本**就是头号候选）；失联真相与断片说的揭示节奏（哪一章哪一拍让玩家知道多少）；会师层细目（灵力系统形态/AI 伴随范围/「回响」命名定夺/造物主层留白深度）归进渊章专门 session；双路平衡（材料经济 vs 精神风险·中途换边代价〔六批已挂〕·「驾驭」分流候选口径）同归。**锁盒一并议（候选·未拍）**：若人渐变像鱼、且死在深处的「你们」都在下面继续往下走——导师日志后半画的「未知鱼类」草图是什么，图鉴的最后一个条目会是谁。

## 5. 写作规约（内容 session 遵守）

1. 每潜点具象人形意象 ≤1，其余幻觉走行为/环境异常（鱼群转向、地形「不对」、声源错位）。
2. 幻觉优先机制型（错误信息：读数/坐标/方向/红字选项），画面型作调味。
3. 现有「越深越稀疏抽象」环境梯度不变；幻觉喧哗度独立上升（§0「环境越空，脑子越满」）。
4. 剧情只走现有通道：事件卡/POI 发现物/物品文本/录音——不加过场动画、不加旁白。
5. 原创鱼命名律：一章=栖所+质感+真实类群（萤光背礁鳚式）；二章=祭祀/蚀刻语感；番外=成对/归途意象。幻觉变体名从原创鱼长出。
6. 深度/氮醉 = 诗意压缩，禁止按真实生理学「纠正」。
7. 一切沉沦/滞留结局必须是玩家选择的结果（无脚本死口径覆盖叙事章节）。
8. 角色可以下结论「是氮醉」，游戏永不交底（叙述文本不裁决真假）。
9. **解释的接力**：每章的「真相」只负责解释上一章的怪，并必须留下自己解释不了的东西——氮醉（一章）→ 天然畸变+氮醉（二章）→ 发达文明（三章）→ 文明自己的未解。**答案是台阶，不是地板。**
10. 主角尸体的脸/身份细节**永远不可见、不可确认**（死亡口径 §3.6）；一二章文本对死亡机制零解释、零暗示。

## 8. 实装路线图（Phase 制·一 Phase ≈ 一个 session·沿深水区 Phase 0–3 的成功节奏）

**总原则**：
- **剧情骑在现有轨道上跑**：事件/flag/POI/lore 四个既有通道承载全部叙事（规约 4），不发明新叙事载体。
- **机制按需长出**：一章先用 `profile.flags` + `lore.*` 跑通，图鉴/录音系统等内容证明需要了再建形状（深水区先例：声呐是被需要逼出来的）。
- **每 Phase 必留 regress 抓手**：加内容必配 scenario baseline（铁律不变）；剧情 flag 链本身就是可断言的状态机。
- **分工**：剧情 beats（锚点链/结局/tells）由交互 session 写；**周末引擎只做氛围变体**（[氮醉] 批次等），不碰锚点链与结局。
- 存档照 quirk #99：story flags 走 additive 套路（#66/#69 先例）不 bump；动了形状就 bump 弃档，不写迁移。
- **优先级（决策 10）**：先把一章做好——St0–St4 为当前主线；St5+ 待一章打磨满意后开，期间后续章节想法随做随记（§7 登记）。**最小第一步=教学部分**（St0：教学关植入半本日志 + story flag 脊柱最小版——教学钩要有处可挂）。
- **回流解锁批（决策 11）**：每章新机制落地后，配一笔老区域回流内容批（capability-gated 选项点亮+解锁高亮·§3.5）；新章内容可提前在老区域埋少量锁点（自然暗示、不 UI 化明示）。
