## 1. 地理 / 动线：三区下行

一条**垂直下行、恐惧递增**的动线；每区挂不同机制。整体因果时间线：**当年是海平面的红树林沼泽兼乱葬岗**（尸体养分喂出茂密红树·生物学真实）→ 文明覆灭 + 涨潮/下沉淹没 → **淹死的红树林**（枯干 + 支柱根迷宫）。所以"茂密"是它当陆地时的化石遗留，不是深水里还在长活树。

1. **树根区**：支柱根迷宫，导航 / 探路 + 尚有透光。入口氛围区。
2. **雾区**：低能见度的悬浮沉积物 / 硫化氢化能菌雾（分解产物·与骨床有因果链）。三区里机制最密：
   - ~~**幻觉实化**（Ch2 主机制）的完美底料——雾里分不清真敌人与低理智幻象。~~ « 2026-07-10 重构（作者拍·见 §0）：雾区改向**诚实的声呐受限迷宫**——雾＝低能见度 + 声呐射程受限（诚实·看不清就是真看不到、不是幻觉），挑战是**导航绕出去解锁后图**；「真敌人 vs 幻象难分」的感知欺骗**删除不重建**（恐吓感改由真迷宫拓扑 + 食腐生态精英怪 + 环境毒物承担）。 »
   - **鬼打墙 / 迷路**：复用既有「迷路」mapgen（蓝洞群那套双向连通图·环/死路/多最深点·`mapShape` 分流·见 maze_mapgen memory），**不发明第二套**。
   - **环境毒物**（水蛭 / 瘴气 / 食腐群）同场。
3. **骸骨区**：最深；真相层 + 最恶遭遇。挂 §7 canon 既有的道德抉择（破坏 / 不破坏尸体 → 进更深区 → 周围尸体尽活 → 更不利围攻）。

**附属 POI（挂在雾区 / 骸骨区里）**：

- **水葬台**（剖尸石台 + 蠕动食腐群·见 §3）。

**「深不见底」保持感知性**：雾 + 浑浊挡住真底，你看见的全是骨、但"无底"是能见度造成的——不用真堆出无限骨量，还扣游戏北极星「越深越欺骗」。

## 2. 差水质食腐生态（精英怪层·全岛共享）

**分层原则**：**精英怪 = 全岛共享的生态底噪**（差水质食物链，哪个区域都可能撞上）；**boss = 各教派 capstone**（见 §5）。

**水化学底座**：大规模弃尸 / 强迫水葬 + 分解 → 缺氧 / 无氧、硫化氢（黑泥、瘴气）、高浑浊（＝雾）、富营养。故生态 = 耐低氧 + 耐硫化 + 食腐者。

### 2.1 舞台层（一眼「这水是死的」，不用解释）
- **白硫菌席**（Beggiatoa 类）：缺氧硫化烂泥上的惨白丝状菌毯，铺满骨床、随流飘。**既遮蔽、也防腐**（抑制清道夫和腐解 → 这也是"骨还在"的原因之一）。配**硫化黑泥 + 硫化氢瘴气**（canon 毒气/瘴气伤）。
- **水母爆发**：生态上正是垂死/缺氧海的标志（死区、过捕必爆水母）。雾里飘满苍白伞，蜇人帘子若隐若现；毒液挂麻痹/错乱（喂 Ch2 精神轴）。

### 2.2 食腐群层（骸骨就是它们造的·"活地毯"）
- **端足类活地毯**（amphipod swarm）：成群几天把尸体啃成白骨——**它们就是骸骨区存在的原因**。画面＝地板本身在蠕动，一动就搅起吃肉的云。演出方向＝木乃伊归来圣甲虫群那种「几秒后只剩骨头」的倒计时压力。机制＝群体持续掉血 / 剥蚀 DoT。
- **食腐等足**（cirolanid isopod）：装甲版活地毯，与端足叠。
- **盲鳗**（hagfish）：黏液 + 钻进尸体从内部吃 + 打结；耐低氧极强。**设计彩头＝假 tell**：盲鳗从一具"尸体"里爆出，误导玩家、把注意力从真正的水鬼 B tell（不腐尸体）引开——两种"尸体会动"叠着玩。

### 2.3 伏击潜伏者层（根隙 / 骨缝里）
- **矶沙蚕 / 博比特虫**（bobbit worm）：埋进烂泥只露钳颚，弹出拖走猎物。可做**招牌伏击精英**（爆发伤 + 拖拽）。**巨型博比特**＝future·见 §5.3。
- **裸胸鳝 / 鳝鱼**（moray / swamp eel）：藏根缝骨缝突袭；血食型挂长毒 / 流血。**鳝鱼选得硬**——耐低氧、能吞空气呼吸，差水质活得最好。
- **水蛭**：canon Ch2 既列（六批），来源见 §3。

### 2.4 状态映射 & 分工 & 护栏
- **状态映射对齐 canon Ch2**（错乱/不良状态/长毒/水蛭/瘴气）：水蛭+盲鳝→流血长毒；瘴气/硫化→毒气；端足/等足群→剥蚀 DoT；水母→麻痹/错乱；博比特→伏击爆发。
- **内容分工**：带机制的（博比特伏击、群体 DoT、盲鳗假 tell）走**交互 session** 建；纯氛围鱼（飘过的水母、根缝小鳝）可一站一种喂**周末引擎**（weekend_content_log 约定）。
- **别混水鬼 B**：这些都是**普通（恶心）生物**——当年吃掉肉、留下骨；B 是"本该烂却不腐"的反常 tell。两种恐怖分开摆。若用带寄生的（如缩头鱼虱 Cymothoa）只让它**寄生鱼、不碰人**——人体寄生锁给髓织虫线，别抢 B 的揭示。

## 3. 水葬设定（天葬式 → 强迫水葬 → 生态更替）

- **原型**：天葬式——剖开腹/背，让鲨鱼吃掉，助食取食（对位天葬剖尸）。原本是厚葬死者的仪式（美化：归海 / 归神）。
- **堕落转折（金句）**：后祭祀文明起头**强迫活人水葬**——从"厚葬死者"堕成"处决活人"，正是 canon「碎裂堕落」的现身，也是一组**美化版 → 真相**。
- **生态更替（真生态）**：数量暴增 + 水质变差 → 鲨鱼（要干净高氧的活跃掠食者）离开/死亡 → 转由**食腐鱼 / 虫**接手：鳝鱼、水蛭、及 §2 的食腐群。方向完全合理（缺氧富营养硫化排斥大掠食者、利食腐无脊椎）。
- **canon 缝合**：给 canon Ch2 既列的**水蛭**补了 in-world 老家，不是空降环境伤。
- **载体**：水葬台做成雾区/骸骨区里的独立 POI 小景（剖尸石台 + 蠕动群），挂一对美化/真相文本。

## 4. 竞技场（教派1 的场所）

- **画面**：超大规模大乱斗——活人在人、猛兽的尸体、甚至奄奄一息的躯体和内脏上厮杀、吞食。
- **纵向逻辑（白送）**：竞技场是每天在坑面发生的事，**几百年的竞技场日 = 底下深不见底的骸骨地层**。骸骨区 = 竞技场的地质记录；动线下行 = 时间回溯。
- **美化版 → 真相**：台面＝神圣的角斗 / 献给神的搏杀；真相＝**在一层活人地毯上的屠宰**。
- **"堆满"体量来源**（诚实账）：决斗 / 献祭 / 吃人只是**配料**；主力是「**整个子文明几百年的死者全汇流到这一处**（唯一合规弃尸/葬地·参照巴黎地下墓穴几百万、玛雅圣井）+ **外来俘虏管线**（劫掠邻近小文明·天然扣"他者恐惧"题眼）+ **崩塌那一波大规模献祭**做最深地层」。
- **吃人 = 为什么是骨不是尸**：肉总被吃掉、只有骨丢进坑——吃人恰好解释了这是**骸骨床**而非尸体床。这条因果别丢。

## 5. 教派结构（boss = 教派 capstone）

**结构原则**：**一 boss 一教派**——每个 boss 具象一个后祭祀文明主要教派。这咬合 canon「后祭祀文明碎成许多小文明·各有信仰」，并与 Ch1「四大 boss = 两派四分支各一」押韵，给禁岛立了套 **boss 分类学 = 教派**。天然复用祭祀文明 SPEC §5「美化版 → 真相」通用模板：**每教派一套**（一个揭示门 + 一份美化经文 + 一份真相记录）。

### 5.2 互黑 = 真相通道（揭示机制·本档核心发现）
两教派各自美化自身、却对**敌人说真话**（恶意驱动的准确）。玩家**交叉比对两边的互黑文本**即可拼出真相——谁也不会美化敌人。扣认识论压平 / 断片说，守"诚实通道"合同，比单纯"美化 vs 真相"多一条现成的路。

- **教派2 → 教派1**：「**总是自相残杀、残暴野蛮**」（表面属实——他们确实互相屠杀）。
- **教派1 → 教派2**（弱肉强食信"强者自己去夺"，故最狠三角度）：
  1. **「行尸 / 被穿的壳」**：「你们不是永生，是**死了还在动、被虫穿着的空壳**——连自己都不是了。」——**恰好戳中水鬼 B 真相**（canon：宿主是死是活无法定义、只剩壳被穿）。这是"互黑=真相通道"最漂亮的一击。
  2. **「懦夫 / 被吃的那个」**：「你们**把女人喂给虫、男人跪求被感染**（扣髓织虫献祭女性·祭祀 SPEC §2.2），连吃都不敢主动，是**被吃的那个**，不是吃的那个。」
