## 1. 病根：丢延续（两派共同的问题）

现实章鱼＝个体极聪明，但**繁殖即死（semelparity）·世代不重叠·知识不跨代累积**＝有智力、没延续（对齐 canon「退化成浅水章鱼＝留智力丢延续＝现实章鱼生物学」）。

整个古文明＝在回答同一个伤口：**个体够聪明，但延续为零，怎么把心智留下来、还叠得更高？** 两派＝这道题的两种解；四子派系＝两种解各自的两条路线。这条把「为什么会分派」从散设定收成一个根。

## 2. 四子派系：三层结构

memory 既有四派（tech / bio 各裂二）＝「怎么跨阈」的四条研究路线。本档给它三层读法（三层共存、互不取代）：

| 派 | 极 | 离头足本性 | 交叉讽刺（表 → 里） |
|----|----|----------|------------------|
| 生物·蜂群 | 集体 | 背离（章鱼本独居） | 表＝社会性顶点；里＝工蜂被夺自由意志、整巢服务单一中心＝**以个体为中心**（超个体：一个个体借上万具身体） |
| 生物·个体 / 弱肉强食 | 个体 | 贴本性 | 表＝纯自利；里＝捕食压力逼出结盟 / 共生 / 分工＝**长出更丰富的社会** |
| 科技·中心化算力 | 集体 | 背离 | 表＝集权；里＝取消私有→算力像空气一样按需分配＝**每个个体拿到更多算力** |
| 科技·去中心化算力 | 个体 | 贴本性（腕足各自算的神经） | 表＝人人平等自持；里＝所有权＝可囤积＝最尖端算力**聚到少数人手里** |

**三层读法：**

- **交叉讽刺（本 session 核心）**：每一极都长出自己的反面；且**哪一极是陷阱，两派正好相反**——生物派的陷阱在集体极（蜂群），科技派的陷阱在个体极（去中心化）。所以主题不是「集体 / 个人谁对」，而是**标签≠实质，且两派对同一个词的判断相反**＝他们为何是两派。推论：**每派的美德出口到对方领域就成恶德**（把「逼出个体→长社会」搬到算力＝私藏；把「取消私有→公地」搬到身体＝蜂群暴政）。
- **离本性（memory 既有）**：去中心化 / 个体贴章鱼本性，算力集中 / 蜂群是背离本性的另投＝给「一派为何裂两支」一个 in-world 由头（顺天性 vs 逆天性两拨）。
- **失败的跨阈（memory 既有·接水鬼）**：四派＝四种想跨过深渊法则的尝试，**四条都失败**＝水鬼「成因多样」的具体四来路（见 §6）。

**母题（把四派收成一句）**：「同一个基底、承载众多」这个结构，蜂群（生物集体）和算力公地（科技集体）是**同一个形状**——一个心智穿众多身体 vs 众多心智跑在同一基底。它在生物侧是暴政、科技侧是救赎，**区别只在众多者保没保住自己的意志**——这正是水鬼那道题（意识转移里丢没丢「我」）。

**算力轴的政治经济学（2026-07-06 补·「初衷 → 结局」的对偶）**：上表两条科技行的「表 → 里」也可读成**时间线上的自我背叛**——去中心化算力 初衷＝人人平等自持（分散·无中枢）→ 结局＝**垄断**（抽掉正式中枢，非正式权力＋规模／网络效应让小玩家坍缩成巨鲸·现实回声：比特币算力聚矿池 / tyranny of structurelessness）；中心化算力 起手＝像集权 / 垄断 → 结局＝**共同富裕**（算力像空气·人人拿到更多）。三点收口，**别读成「中心化好」**：

1. **精度**：「垄断 → 共同富裕」只在垄断＝**几个互相压价的巨头，或一个可问责的公共池**时成立；单一、不受制衡的垄断者只会抽租。真正给出富裕的是**可竞争性 / 可问责性**（power contestability），不是「集中」本身——这也正是去中心化翻车的同一机制（它把制衡一起抽掉了）。隐藏变量＝「权力可不可被挑战」，中心化只是**碰巧更容易保住它**；两极端都塌（全分散因规模重新聚顶·单一中枢就是字面垄断），可竞争的**寡头中段**才稳。
2. **第三道反转（守本节「不选边」+ 扣北极星）**：共同富裕是真的，但**同一份富裕就是笼子**——「算力像空气、人人都有」＝全民把认知外包给深渊那口算力池＝依赖到拔不掉＝正是「过高等不能上浮」（实例化 §5a）。所以不是「中心化对」，是**它的礼物就是毒**；讽刺落在恐怖上、不落在政治结论上。（与母题正交：母题那道是「转移里丢没丢我」，这道是「富裕带来的上浮枷锁」，两条边各切一刀。）
3. **两条机制在价格 → 0 处合流**：寡头压价把算力打到近零（§3 湿件靠热液能量近免费·边际成本趋零），边际成本一旦归零，「市场买」与「按需拿」不可区分＝资本路与共产路**走到同一终点、成同一个东西**（又一次标签≠实质·顺手化掉「上表中心化里读法用市场竞争还是用按需分配」的机制二选一）。

**接 §6**：这套分配把「谁能用好基底跨阈」变成经济问题——被垄断挡在门外者只能拿劣质基底硬跨 → 半成品，是 §6「众多绝望殊途同归」里的一条**经济来路**（不新增派系·只给既有失败尝试补一层压力）。

**载体纪律**：「资本主义／共产主义／共同富裕」等词只作**人类队投上去的标签**（认识论压平·同「克苏鲁」「水鬼」命名），in-world 头足类不直接用；尤其「共同富裕」是很重的人类口号，别塞进古文明口中。

## 3. 科技派技术树：湿件（不碰火）

**决策（2026-07-06 锁）**：科技派的「算力」＝**生长出来的活神经基底**，不是硅基芯片（硅基被物理否决，见 §5b·作用域＝古文明的水生起源）。三阶：

1. **体外记忆**：头足类本用变色皮肤打图案通信；养一种能锁存这些信号的活组织（珊瑚 / 海苔式群体生物）＝第一块「硬盘」→文化第一次跨代累积（直接解「丢延续」）。
2. **体外算力（「算力礁」）**：能存就能改——培育会*变换*信号的神经组织＝他们的计算机；材料是湿神经非硅，又热又耗能，正好吃深海的冷（现实类比：微软 Project Natick 海底数据中心；水的对流散热比空气高一两个量级），供能走热液喷口的化能 / 温差（连供电都不用火）。这一层＝「靠技术**强化**智能」。
3. **搬进去（upload）**：终局＝把自我拷进不死的礁、脱离会死的肉体＝完整意义的「靠技术而非肉体」。

注：这条「把心智外置」的 MOVE 与人类科技派 **Iskra「意识融入 AI·身体消亡」完全同构**（见 §7 镜像）——区别只在基底（古文明只能长湿件；人类从水面带下硅基 AI）。

## 4. 算法层：基底无关，近人类信息科学

**计算是基底无关的**（Church–Turing）：信息论、可计算性、复杂度、纠错、学习，神经也好硅也好同样成立。→ **人类信息科学在他们那套湿件上照样适用；科技派人类角色背景可为计算机 / 信息科学**，一眼认出「这是在编码 / 纠错 / 学习」。

但**架构错位**（recognizable but rotated＝weird-horror 的「认得出 + 不对劲」）：湿的、生长的基底天然是**模拟·超并行·连接主义·无软硬件之分**——「程序」就是组织怎么长 / 怎么连，改程序＝重新培育，更像训练神经网络而非写代码。→ 一眼看穿的多半是**搞机器学习那种人**，不是搞系统 / 写代码那种。

**政治即架构**：湿基底天生分布式，「中心化 vs 去中心化算力」不是外挂的经济政策，而是字面上怎么布线（一张大池化网 vs 许多私有网）＝把 §2 的母题又收一道。

**能力落差（好料）**：湿件慢 / 吵 / 省能 / 皮实，强在海量并行的感知·预测（像大脑 / 大模型），弱在快速精确的串行算术；理论**重叠但旋转**——模拟动力学上深、离散逻辑与形式证明上天真。人类会发现他们的「计算机」算加减蠢得离奇、却能对整片海做出瘆人的预判。

## 5. 深渊法则的物理理由 + 硅基否决

### 5a. 为什么「过高等不能上浮」（机制化）

托着高等心智的基底（蜂群集体 or 算力礁）是**吃深渊的**：靠低温、高压、热液能量、只有深渊养得起的巨量活组织。往浅带→基底撑不住→叠上去那层跑不动。这**实例化**了 memory 既有原则「advancement＝枷锁·要离开必须变简单·信息上浮必先简化→真相升上来必掉成神话」（简化＝褪掉深渊依赖的增强层）。两条出路：

- **顺从法则**＝放掉增强层、退回独居会死的聪明章鱼（留智力丢延续）＝lawful exit。
- **硬抗法则**＝把自我强行拷到一台带不动它的临时破基底→半成品＝**水鬼**（见 §6）。

### 5b. 硅基为何被物理否决（作用域：古文明水生起源，非人类）

造芯片＝**干燥 + 洁净 + 真空 + 高温 + 纳米稳定**，海水逐条反着来：洁净（fab ISO Class 1 vs 海水胶体汤，差十几个数量级，落一粒尘＝废片）；光路（DUV 受控空气 / EUV 真空 vs 海水折射率随盐度温度压强乱漂 + 气泡 + 微生物）；纳米稳定（milli-K 控温 + 隔振地基 vs 流动海底，散热易≠控温易）；盐水导电腐蚀短路电子；上游整树要火 / 真空（~1900°C 电弧炉、1414°C 拉单晶、真空注入 / CVD / 刻蚀）——水下无明火。浸没式光刻≠泡海里（那是镜头-硅片间一层超纯去气控温薄水膜，提数值孔径 1.0→1.44）。

**作用域**：此否决只针对**古文明在水下自研硅基**。人类陆地起源、有火、造得出硅基 AI，再带下水（＝Iskra 融入的那台 AI）。所以「湿件 vs 硅基」＝古文明 vs 人类两条技术路的分野，不是「水里不能有硅」。物理已替设定圆好古文明这条约束，别再回头纠结。

## 6. 水鬼：多向罚则（重申既锁 canon + 本档补机制）

**既锁（2026-07-04·别改）**：水鬼不是物种、是「跨阈失败态」统称、成因多样、看不出来向；设定内授权敌谱多变体；命名本身＝人类又一次认识论压平（同「克苏鲁」）；揭示锁三章。

**本档补的机制层**：水鬼由**失败的形状**定义，不由凶手定义——意识在转移里丢了「我」、只剩壳被穿。**任何一种走到这步的绝望都通向它**，具体多来路：

- 四子派系四条失败的跨阈尝试（§2 第三层）＝**古文明侧的四来路**。
- 人类侧：Voss 生物派**自我实验推太快**→水鬼（唯一玩家知来路的具名者·剧情 SPEC）；想上来的深渊者**心急**→半成品。
- 玩家自己走在「下沉 + 吞古技」线上→水鬼＝**玩家潜在下场的镜子**（扣无脚本死）。

单一成因＝待解的谜；众多绝望殊途同归＝这个世界的**一条法则**（深渊法则的罚则本身）＝更吓人、更统一。
