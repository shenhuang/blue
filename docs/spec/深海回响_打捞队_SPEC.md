## 母组织 · 赤喉海盗团 / The Crimson-Throats

- **形象**：一伙下海捞宝的海盗团，徽章是**军舰鸟**（frigatebird·雄鸟血红喉囊＝"红喉/赤喉"由此来）。黑背鸥小队是其中一支侦察小队，后来**理念不合退出**（小队要探、大团要抢）。
- **军舰鸟徽的用意**：军舰鸟是唯一**下不了水**的海鸟（脚小、羽不防水、沾水就飞不起来）——统治天空、被挡在海面之外，只能掠夺别的鸟或掠食海面为生（"man-o'-war / 海盗鸟"绰号即由此）。一支**下海**的队伍偏拿它作徽＝反讽（"最完美的海盗碰不到我们要去的地方"）+ 挽歌（那只从不沾水的鸟安然无恙，被水吞掉的却是这支下了水的队·Sela 还挂着它的旗）。雄鸟求偶把喉囊鼓成血红气球＝"炫耀战利品"的姿态，也贴打捞队的做派。
- **红喉鹈币（`item.deep_token`）**：赤喉海盗团自铸的**分赃筹码**，边上压着军舰鸟徽。队没了、币还在水里流通。Sela"只认这个"不是规矩，是**除了这个她算不清别的账了**（呼应她的人格轴）。玩家手里的币多是从他们留下的东西里翻出来的。
- **用字**：中文刻意留双词——团名"**赤喉**"、币/鸟"**红喉鹈**"，指同一种鸟、不强行统一（作者拍）。

## 成员详情

### Sela（幸存者·港侧商人·**已实装**）
**名/性别**：Sela·女（希伯来"岩/礁"——像礁石留在浅处·**没真正押边**·故还回得来）。
**押边**：无（既没全押科技也没全押生物→四人里唯一每次都游得回来的）。
**声音**：冷、简，像在报点货；话里全是"东西"、很少提"人"——「我不找人。人会变、会走、会说谎」是**反话**（人全没了才这么说）。有明确来处，但不主动提。
**背景**：黑背鸥小队测绘/侦察一员、赤喉海盗团出身，退出后带小队下海。如今队只剩她。
**动机（关键·从寻宝转向找人）**：她不再为宝藏，为**找回同伴留下的东西**——收藏家。藏品对玩家是废料、对她是仅存的、还记得那三个人的证据（"对玩家无价值、对她有价值"落成机制＝收藏品 class·藏宝 SPEC §12.5）。
**机制角色**：港侧特殊商人（红喉鹈币 + 信任档双门控货架）+ **渐深遭遇**（每解锁更深一根深度柱就在更深处再遇她一次·她只会更深、停不下）+ **三条同伴线的发布者**（档1 Corin / 档2 Nerea / 档3 通向 Vera 的失联线）·信任三档沿用 `[10,30,60]`。
**警示镜**：一个为"物"而非"人"下来的探险家，深海把她磨成什么样；信任越高越露出她比表面更"下不来"的一面（与玩家执念同构）。
**已实装**（#246）：改名（silas→sela 连 id）+ 改女性 + retheme 落地；对话 root/banter/deeper/confession + 新 `sela.corin_offer`/`sela.corin_return`；Corin 线端到端。**（#247）渐深遭遇**：热液区 `roam.sela_meet_deep_*` 深层再遇（比中层更深·置 `flag.sela.met_deep`·供二章 Nerea offer 门控消费）。

**命运**：装备在深处失效→没了（"深海不原谅机械"）。遗物＝**测绘图/数据日志**＝"好图"来源①（Corin 生前钉下的坐标·比通用图更深更好）。
**收藏品（交还 Sela）**：**红喉鹈徽章**（`item.keepsake.corin_badge`·赤喉海盗团的军舰鸟徽记·别在队伍装具上的那种·背面刻着科林自己的记号·花不出去、对玩家废铁、对 Sela 是他的证据）。**徽章两段揭示（作者拍·2026-07-02）**：交还 Corin 徽章（Ch1·stage 1）Sela 认出、收口、只说「说来话长·不是今天」；交还 Nerea 徽章（Ch2·stage 2）她才透露一些赤喉海盗团的旧事——徽章＝「他们从前不只是下海捞宝」的实证，逐件累积揭示母组织。〔原 draft 遗物"损坏的深度表/烧毁的数据模块"仍作为"泡烂了留不住"的东西出现在开箱叙述里〕

### Nerea（生物派死者·draft 名·**整条线移到第二章**·作者拍 2026-07-02）
**押边**：生物——把命押在改造自己上。
**命运**：死于**改造反噬或被吞**。遗物＝**改造手记/变异标本**＝引向深料点/生态点②。
**伏笔（重要）**：Nerea 的"生物改造过度"正是 **Vera 两个解释之一**——生物派的终点埋在这条线里·读完为 Vera 的模糊埋伏笔。
**收藏品**：也是一枚**赤喉海盗团徽章**（`item.keepsake.nerea_badge`·category 'other'·二章建）——交还＝徽章两段揭示 **stage 2**：Sela 借它交底赤喉海盗团旧事（见 Corin 条）。
**落位（作者拍 2026-07-02）**：Nerea 的藏宝点在**第二章**——二章正是**生物派的主要地区**（贴她的生物押边）。二章目前**纯 SPEC 零代码**，故 Nerea 藏宝线（图→藏点→交还 stage-2）**延后到二章建**、不在 Ch1 落。Ch1 已埋的钩：`flag.sela.met_deep`（渐深遭遇·见 Sela 条）留给二章 Nerea offer 门控消费。

### Vera（失联者·**延后建·design-only·别进第一期**）
**名/性别**：Vera（拉丁"真"）·性别**刻意模糊**。
**押边**：押得最狠·**过了线**（彻底押一条路＝那条不可逆的单程下行）。
**人设（处处与 Sela 反着来）**：Sela 还数东西、还攒币、还在找他们、还每次都游得回来；Vera 把这些全放下了——过了线、账不再有意义、不再算、不想（或无法）被找回、静在最底、接受了。Sela 是"快没了"，Vera 是"已经过去了"。
**诡异不可辨（核心·作者拍）**：你终于在最深处见到失联的同伴，但游戏**永不确认那到底还是不是本人**——(a) 死了、被有一定智慧的尸衣者穿上；(b) 生物改造过度、活着但变得和尸衣者一样，**没人能证实**。Vera **不说话**（锁死模糊·也省整棵对话树·靠摆货/手势交易）；玩家一直在她身上找"是她"的证据（按断片说·脸永不可见），game 一次都不给。
**主题（作者深化）**：从人类视角＝"越深越欺骗"；从深海/自然视角或许"越深越真实"——两视角都留、game **永不裁决**（别说教）。它顺带软化 Sela 结局：人这边看是被骗着毁掉（悲剧），深海那边看是被剥回真身（诡异归乡），玩家两种都能读。
**唯一特例**：能开店、有智慧的尸衣者比"顶级捕食者"高一档→定成**全局唯一特例**（更古老/唯一一个）·别让普通尸衣者沾这智慧（守怪物平时威胁·接 `深海回响_敌人库_SPEC.md` / boss 设计的尸衣者定位）。
**不可逆价**：过线后**红喉鹈币失效**（分赃筹码只在"还回得来"的世界算数）；Vera 的"价"＝拿不回来的东西（一次性代价·机制形态待作者·延后设计）。
**依赖**：住"无法返回之地"（最深区·地图落位作者"以后再说"）→ **第一期不建**。
**科考队接口**：作者曾问"失联那个接成导师/科考队第八人"——**已否**（会破坏本队一二章零剧透 + 把边缘 NPC 焊进承重主线 canon + 破坏科考队 7 人对称）；留"擦肩"淡呼应可，硬接不做。

## tech/bio 押边轴（世界观规则 · 诚实边界）

- **世界规则（叙事）**：能下到最深的人，要么押"科技"（外挂装备）、要么押"生物"（改造自身）。**彻底押一条路本身就是那条不可逆的单程下行**——解释了为什么 Sela（没押）还回得来、另外三个（押了）过了线。
- ⚠️ **诚实边界（防空中楼阁）**：**代码里现无科技/生物玩家流派系统**（仅有独立装备升级树 + capability 标）。本队 tech/bio 目前**纯叙事层背景**（两死者一科技一生物·Vera 押边成谜）·**零代码**。真做成玩家可选流派＝独立大工程（`profile.techBioPath?: 'tech'|'bio'` additive 不 bump SAVE + 新 Condition `hasPath` 补 3 switch + 两条装备/改造线 + 大量内容门控·Opus·high·**单列 initiative·别混进本队交付**·藏宝 SPEC §12.7）。

## 主题 · 与科考队的对照

> **别读成两队有关系**——它们叙事上互不相干（见顶部硬约束）；这里记的是**设计层对照/镜像**，供写作口径统一，不在游戏内点破。

- **蓝领镜 vs 机构镜**：黑背鸥小队＝为钱下海的**打捞队**，是主线**科考队**（为求知的机构队）的对照面。两队各自"一支队被深海一个个收走"，母题同构（科考队"失联＝单程下行"）。
- **押边即单程**：科考队用"科技派/生物派"分线（Iskra/Vesper·Voss/Alethea）；打捞队用同一条 tech/bio 轴（Corin 科技死·Nerea 生物死·Vera 押到过线）——**两队共享"押边＝不可逆"的深渊代价观**，打捞队是**平民化、非机构**的那一面。
- **Sela ↔ Vera**（队内镜像）：还在数 vs 已放下；"快没了" vs "已经过去了"；每次游得回来 vs 静在最底。玩家从 Sela 一路见她越走越深，Vera 是这条线的**终点像**。
- **收藏家母题**：Sela 把队友遗物当仅存证据收着——"对玩家废料、对她无价"。可与海洋"囤积/藏物"的生物意象（如章鱼的"花园式"囤积、军舰鸟徽的炫耀姿态）暗合，但这是**Sela 的人格**、不外显成设定说明。
